The prototype is deployed on Streamlit, and can be run by using Jupyter Notebook.
- Find the Section: Streamlit Deployment and run the last code
- Before run code, please make sure all the files submitted are within the same folder with the BookRecommenderSystem.py

Reminder: Install Streamlit in the Anaconda Navigator first
1. Click on 'Environments' at the left panel
2. Create a virtual environment (Create button at the bottom)
3. Fill in the name and select python, then create
4. Select the new created environment
5. Filter the installed to not installed at the drop down menu
6. Search 'streamlit' & 'streamlit-extras' and download it
7. Open the Anaconda Prompt
8. Type: conda create --name streamlit_newenv python=3.13
9. Type: conda env list 
-> to check whether the new environment has been successfully installed or not
10. Switch to the new environment: conda activate streamlit_newenv
11. Type: pip install streamlit
12. Type: pip install streamlit_extras
